제작동기


 홍콩 과기대의 김성훈 교수님의 모두를 위한 머신러닝/딥러닝 강의를 들으면서 강의 노트가 있으면 좋겠다는 생각이 들어서 직접 만들게 되었습니다.
TIP. 기본적으로 27인치 모니터 기준으로 작성하였습니다. 화면을 반씩 쪼개어 한쪽은 강의를 다른 한쪽은 강의 노트를 놓고 보시면 편합니다.

  이론 강의와 TensorFlow 코드 강의를 왼쪽 side bar에 나눠 놓았습니다.
강의를 들으시면서 의문점이 있거나 TensorFlow에 대해 좀 더 공부를 하고 싶으시면 reference란의 TensorFlow KR 팀 공식 페이스북 페이지 타임라인에 질문을 남기시거나, 한글로 번역 된 공식 TensorFlow API document를 참조하시기 바랍니다.

 본 강의 노트의 code syntax는 visual studio 2015 스타일로 작성 되었습니다.
만약 code syntax 스타일을 바꾸고 싶거나 강의 노트를 수정하고 싶으시면 저에게 연락을 주시기 바랍니다. ( html5, javascript 에 해박하신 분은 직접하시면 됩니다~ )
